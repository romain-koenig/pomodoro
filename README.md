# 🍅 Pomodoro Timer

A clean, fully-featured Pomodoro timer app hosted as a GitHub Page.  
Inspired by [pomofocus.io](https://pomofocus.io/).

## Features

- **Three timer modes**: Pomodoro (25 min) · Short Break (5 min) · Long Break (15 min)
- **Start / Pause / Reset** controls
- **Session counter** — tracks completed Pomodoros (persisted in `localStorage`)
- **Audio notification** — triple beep via Web Audio API when the timer ends
- **Task list** — add, check off, and delete tasks (persisted in `localStorage`)
- **Dynamic tab title** — shows current countdown and mode in the browser tab
- **Colour-coded themes** — background changes per mode (red / teal / blue)
- **Responsive design** — works on desktop and mobile

## Usage

Open `index.html` in any modern browser, or visit the GitHub Pages URL.

## GitHub Pages

The app is deployed automatically from the `main` branch via GitHub Pages.
