Politique de confidentialité

En bref : Votre vie privée est importante pour nous. Ce site ne collecte aucune donnée personnelle et n'utilise aucun service de suivi. Toutes vos données (tâches, paramètres) restent stockées localement sur votre appareil.

1. Introduction

La présente politique de confidentialité décrit comment le site pomodoro.kromatic.fr (ci-après « le Site ») traite les données des utilisateurs. Cette politique est conforme au Règlement Général sur la Protection des Données (RGPD - Règlement UE 2016/679) et à la loi française Informatique et Libertés.

3. Données collectées

3.1 Données stockées localement

Le Site utilise le stockage local du navigateur (localStorage) pour sauvegarder :

  • Vos tâches : la liste des tâches que vous créez et leur état (complétées ou non)
  • Vos paramètres : les préférences de minuterie, les paramètres de son, et autres configurations
  • Votre progression : le nombre de sessions Pomodoro complétées

Important : Ces données sont stockées exclusivement sur votre appareil (dans votre navigateur) et ne sont jamais transmises à nos serveurs ou à des tiers. Nous n'avons aucun accès à ces informations.

3.2 Données de connexion

Le Site est hébergé sur GitHub Pages. Conformément à leur politique, GitHub peut collecter des données de journal techniques limitées lors de l'accès au site, notamment :

  • Votre adresse IP
  • Le type de navigateur utilisé
  • La date et l'heure de la visite
  • Les pages consultées

Ces données sont collectées et gérées par GitHub conformément à leur politique de confidentialité.

4. Absence de cookies tiers et de suivi

Le Site n'utilise :

  • Aucun cookie de suivi
  • Aucun service d'analyse (Google Analytics, etc.)
  • Aucun pixel de suivi publicitaire
  • Aucun service de mesure d'audience tiers

Votre navigation sur le Site est totalement privée et non suivie.

5. Finalité du traitement

Les données stockées localement sur votre appareil ont pour unique finalité :

  • De vous permettre de sauvegarder vos tâches entre différentes sessions
  • De conserver vos préférences et paramètres personnalisés
  • D'améliorer votre expérience utilisateur

6. Durée de conservation

Les données stockées dans le localStorage de votre navigateur sont conservées indéfiniment jusqu'à ce que vous :

  • Supprimiez manuellement les données via les paramètres de votre navigateur
  • Effaciez l'historique et les données de navigation de votre navigateur
  • Supprimiez les tâches ou réinitialisiez les paramètres depuis l'application

7. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : vous pouvez consulter vos données via les outils de développement de votre navigateur (localStorage)
  • Droit de rectification : vous pouvez modifier vos tâches et paramètres directement dans l'application
  • Droit à l'effacement : vous pouvez supprimer toutes vos données en effaçant le localStorage de votre navigateur ou en supprimant les données du site dans les paramètres du navigateur
  • Droit à la portabilité : vous pouvez exporter vos données en utilisant les outils de développement de votre navigateur

Pour exercer ces droits ou pour toute question concernant vos données, vous pouvez nous contacter à l'adresse indiquée en section 2.

8. Sécurité des données

Bien que les données soient stockées localement sur votre appareil, nous vous recommandons de :

  • Maintenir votre navigateur à jour
  • Utiliser un appareil sécurisé
  • Ne pas partager l'accès à votre appareil avec des personnes non autorisées

9. Liens externes

Le Site peut contenir des liens vers des sites externes (par exemple, kromatic.fr). Nous ne sommes pas responsables des pratiques de confidentialité de ces sites tiers. Nous vous encourageons à consulter leurs politiques de confidentialité respectives.

10. Modifications de la politique de confidentialité

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Toute modification sera publiée sur cette page avec une date de mise à jour. Nous vous encourageons à consulter régulièrement cette page pour rester informé de nos pratiques en matière de confidentialité.
